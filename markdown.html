マークダウン記法

  # は見出し
  ###### ６個まで使える。
  
  - 見出し　とすると・見出しになる。リストとなる。+や*でもなる。
  
  > とするとQAサイトのように引用のマークとなる。
  
  --- とすると区切り線。前後に必ず空行が必要。
  
  文字の前後に*を入れると斜体になる。**を入れると太字になる。
  
  連番を入れるには
  1. 動画を見る
  1. コードを書く
  1. 改造する
  このようにする。
  
  これで
  1. 動画を見る
  2. コードを書く
  3. 改造する
  となる。
  
  - [ドットインストール](https://dotinstall.com)
  とすると、文字にリンクが設定される。
  
  ソースコードを入力する前後の行に
  バッククォートを3つ入れる。バッククォートはShift+@。
  
  ```
  function f() {
    return 0;
  }
  ```
  
  文字列の中に入れる場合は
  特に`return 0;`の意味がわかりません。
  と前後を囲む。
