HTMLのよく使うタグ

タグ 説明 例
h1 見出し。H1はheading1という意味。 H1～h6まで。 <h1>見出し</h1>
p paragraph=段落 <p>段落です</p>
img 画像を表示 <img src="image.png" alt="例の画像です" width="160" height="160">
style CSSタグ <head>タグ内や別ファイルに記載する。 
        <style>
          img {
             border-radius: 50%;
             }
          section {
            border-width: 1px;
            border-color: gray;
            border-style: solid;
            border-radius: 8px;
            padding: 16px;
            margin-bottom: 8px;
            background-image:url(back.png);
          }
          body {
            text-align: center;
          }
        </style>
footer フッターを表す。末尾じゃなくても実はOK。 <footer>© hogehoge.com</footer>
section 意味のあるかたまりに対して汎用的に使えるタグ。スタイルシートで指定する時に便利。 これをつけても見た目に変化はない。 <section>XXX</section>
html VS Codeエディタではhtml:5 もしくは!でEnterとするとひな形が自動で入る。 
          <!DOCTYPE html>
          <html lang="ja">
          <head>
            <meta charset="UTF-8">
            <meta http-equiv="X-UA-Compatible" content="IE=edge">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Document</title>
          </head>
          <body>
            
          </body>
          </html>
コメント VS Codeエディタでは Ctrlキー＋スラッシュ（/）とする。 <!-- コメント -->
ul 箇条書きリスト Unordered List 
        <ul>
          <li>HTML</li>
          <li>CSS</li>
          <li>JavaScript</li>
        </ul>
ol 順序付きリスト Ordered List 
        <ol>
          <li>HTML</li>
          <li>CSS</li>
          <li>JavaScript</li>
        </ol>
dl 説明付きリスト Description List Description Details 
        <dl>
          <dt>HTML</dt>
          文書の意味
          <dt>CSS</dt>
          文書の見た目
          <dt>JavaScript</dt>
          文書の動き
        </dl>
pre HTMLタグをそのまま表示するためのタグ 
        <pre>
          &lt;p&gt;HTMLタグをそのまま表示しようと試みる&lt;/p&gt;
          &lt;p&gt;pタグでマークアップしたテキスト&lt;/p&gt;
        </pre>
table 全体をかこう 
        <table>
          <thead>
           <tr><th>。。。</th><th>。。。</th></tr>
          </thead>
          <tbody>
           <tr><td>。。。</td><td>。。。</td></tr>
           <tr><td>。。。</td><td>。。。</td></tr>
           </tbody>
         </table>
thead 見出し部分 テーブルヘッド
tbody 本体 テーブルボディ
tr 行 table row
th テーブルヘッダー
td テーブルデータセル
a アンカータグ <a href="https://256times.com" target="_blank">256times</a>
aパート２ ページ内にとぶ場合はidを使用。id側には#をつけないことに注意。 
        <a href="#lessons">とびます</a>
        <section id="lessons">。。。</section>
nav ナビゲーションタグ sectionと同様、文書の区切りをわかりやすくするだけでブラウザで表示は変わらない。
header ヘッダータグ <header>。。。</header>
small 著作権情報など注釈的に小さく表示したい情報を表現するために使用
title ブラウザのタブやブックマークしたときのタイトル、検索結果で表示されるときに使われる
ファビコン ブラウザのタブやブックマークでタイトルの横に表示されるアイコンのこと 
        <head>
          <link rel="icon" href="favicon.png">
        </head>
main 使わなくてもよい
aside 広告などの段落をわかりやすくする意味で使用。
article 記事のようなコンテンツ sectionと同じく文書の区切りを明確にするだけ。見た目は変わらない。
br 改行。閉じタグがない。
&lt; <
&gt; >
code 文中でコンピュータプログラムのコードを表現する際はcodeタグを使うと意味が明確になる。
span 文中の一部にスタイルを付けたいときに使う。1文字だけ赤くしたいなど。 <h3><span style="color:red;">新</span>機能ができました！</h3>
div 文中の一部ではなくて一つ以上の要素をまとめたブロックをスタイリングしたい場合に使う <div style="margin-left:40px;"> 。。。</div>
